# Data Sources

Sources utilisées pour les données scientifiques du BioDex.

## Méthodologie

Les données scientifiques ont été générées en utilisant **Gemini en mode recherche approfondie** le 23 avril 2026 pour garantir l'exactitude et la fiabilité des informations. Cette approche a permis d'intégrer les mises à jour les plus récentes de l'IUCN (2024) et de valider la taxonomie moléculaire pour certaines espèces.

## Source Principale

- **Gemini (Google AI)** - Recherche approfondie en mode scientifique
  - Date : 23 avril 2026
  - Couverture : 31 espèces du BioDex
  - Validation croisée avec IUCN Red List 2024

## Sources Générales de Référence

Les données générées par Gemini ont été validées et croisées avec les sources suivantes :

- **IUCN Red List** - Statuts de conservation (mise à jour 2024)
  - https://www.iucnredlist.org
  - Utilisé pour tous les statuts IUCN
  - Reclassement noté : Hérisson européen (LC → NT en 2024)

- **GBIF (Global Biodiversity Information Facility)** - Distribution géographique
  - https://www.gbif.org
  - Validation des noms scientifiques

- **Encyclopedia of Life** - Informations taxonomiques
  - https://eol.org
  - Nomenclature binomiale standard

- **FishBase** - Espèces marines
  - https://www.fishbase.org
  - Espèces : Poisson-clown, Demoiselle bleue, Poisson-papillon, Hippocampe, Tortue verte

- **BirdLife International** - Espèces aviaires
  - https://www.birdlife.org
  - Espèces : Martin-chasseur pygmée, Coua bleu, Chouette chevêche, Huppe fasciée

## Corrections Taxonomiques Importantes

1. **Chouette chevêche malgache** : Reclassée de *Ninox* à *Athene superciliaris* par analyse moléculaire (gènes cytochrome b et RAG-1)
2. **Osmie rousse** : Correction de *Osmia rufa* à *Osmia bicornis*
3. **Syrphe ceinturé** : Correction de *Syrphus ribesii* à *Episyrphus balteatus*
4. **Gecko diurne** : Correction de *Phelsuma sp.* à *Phelsuma laticauda*
5. **Abeille noire malgache** : Correction de *Apis mellifera mellifera* à *Apis mellifera unicolor* (sous-espèce endémique)

## Notes

- Toutes les données ont été validées par des sources scientifiques reconnues
- Les noms scientifiques suivent la nomenclature binomiale standard
- Les statuts IUCN correspondent aux évaluations les plus récentes (2024)
- Deux espèces supplémentaires ont été identifiées par rapport au BioDex initial :
  - *Liotrigona bitika* (micro-abeille sans dard)
  - *Apis mellifera ligustica* (abeille mellifère italienne)
