# IUCN Status Reference

Référence des statuts de conservation IUCN (Union Internationale pour la Conservation de la Nature).

## Codes IUCN

| Code | Label (Français) | Label (Anglais) | Description |
|------|------------------|-----------------|-------------|
| EX | Éteint | Extinct | Espèce disparue à l'état sauvage |
| EW | Éteint à l'état sauvage | Extinct in the Wild | Existe uniquement en captivité |
| CR | En danger critique | Critically Endangered | Risque extrêmement élevé d'extinction |
| EN | En danger | Endangered | Risque très élevé d'extinction |
| VU | Vulnérable | Vulnerable | Risque élevé d'extinction |
| NT | Quasi menacé | Near Threatened | Proche d'être menacé |
| LC | Préoccupation mineure | Least Concern | Risque faible d'extinction |
| DD | Données insuffisantes | Data Deficient | Informations insuffisantes |
| NE | Non évalué | Not Evaluated | Pas encore évalué |

## Utilisation dans le BioDex

Les espèces du BioDex utilisent les codes IUCN suivants :
- **LC** (Least Concern) - Préoccupation mineure
- **VU** (Vulnerable) - Vulnérable
- **NT** (Near Threatened) - Quasi menacé
- **CR** (Critically Endangered) - En danger critique
- **EN** (Endangered) - En danger

## Sources Officielles

- IUCN Red List: https://www.iucnredlist.org
- Base de données consultée pour validation scientifique

## Notes

- Les statuts IUCN doivent être régulièrement mis à jour selon les évaluations officielles
- Certains statuts peuvent varier selon les sous-espèces ou populations géographiques
