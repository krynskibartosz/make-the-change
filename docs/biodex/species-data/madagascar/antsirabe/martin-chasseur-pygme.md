# Martin-chasseur pygmée

## Informations de Base

- **Nom français** : Martin-chasseur pygmée
- **Nom scientifique** : Corythornis madagascariensis
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Antsirabe (Madagascar)

## Description

Petit oiseau forestier très coloré et territorial. Bien qu'il tolère une légère fragmentation, il nécessite une couverture arborée pour chasser efficacement.

## Habitats Naturels

- Forêts pluviales
- Lisières forestières
- Zones ripariennes
- Forêts secondaires

## Menaces Principales

- Fragmentation de l'habitat
- Expansion agricole
- Pression de chasse locale

## Lien avec la Biodiversité Locale

Il soutient le réseau trophique aviaire en tant que prédateur de premier ordre ciblant les gros invertébrés et les petites grenouilles.
