# Charançon girafe

## Informations de Base

- **Nom français** : Charançon girafe
- **Nom scientifique** : Trachelophorus giraffa
- **Statut IUCN** : NE
- **Statut IUCN label** : Non évalué
- **Projet associé** : Antsirabe (Madagascar)

## Description

Insecte spectaculaire au dimorphisme sexuel prononcé, le mâle possédant un cou allongé pour les combats. La femelle plie habilement les feuilles pour protéger son œuf.

## Habitats Naturels

- Forêts pluviales
- Lisières forestières
- Arbustes endémiques
- Canopée

## Menaces Principales

- Perte d'habitat
- Déforestation

## Lien avec la Biodiversité Locale

L'incubation dans les feuilles enroulées accélère la sénescence végétale, facilitant le recyclage rapide des nutriments dans l'écosystème détritique.
