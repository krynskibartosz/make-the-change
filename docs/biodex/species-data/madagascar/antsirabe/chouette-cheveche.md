# Chouette chevêche

## Informations de Base

- **Nom français** : Chouette chevêche
- **Nom scientifique** : Athene superciliaris
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Antsirabe (Madagascar)

## Description

Rapace nocturne endémique à la taxonomie complexe, récemment reclassé par analyse moléculaire. Il chasse activement dans les milieux semi-ouverts.

## Habitats Naturels

- Forêts sèches de l'ouest
- Forêts humides
- Zones rocheuses
- Milieux semi-ouverts

## Menaces Principales

- Perte d'habitat
- Déforestation
- Fragmentation

## Lien avec la Biodiversité Locale

Prédateur nocturne apex, elle limite les populations de rongeurs et de grands invertébrés nocturnes, évitant la destruction des réserves de graines.
