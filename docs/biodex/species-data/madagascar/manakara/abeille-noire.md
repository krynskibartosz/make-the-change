# Abeille noire

## Informations de Base

- **Nom français** : Abeille noire
- **Nom scientifique** : Apis mellifera unicolor
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Manakara (Madagascar)

## Description

Sous-espèce endémique d'abeille malgache, essentielle aux réseaux de pollinisation sauvage et aux économies apicoles locales. Elle garantit le succès reproducteur des angiospermes endémiques malgaches, formant la base du réseau trophique forestier.

## Habitats Naturels

- Forêts pluviales
- Lisières forestières
- Zones agricoles
- Plantations
- Jardins tropicaux

## Menaces Principales

- Perte d'habitat
- Exposition aux produits agrochimiques
- Épuisement des ressources florales
- Changement climatique

## Lien avec la Biodiversité Locale
