# Olivier

## Informations de Base

- **Nom français** : Olivier
- **Nom scientifique** : Olea europaea
- **Statut IUCN** : DD
- **Statut IUCN label** : Données insuffisantes
- **Projet associé** : Sardaigne (Italie)

## Description

Arbre fondateur du maquis dont la variété sauvage (oléastre) recèle un riche patrimoine génétique. Il souffre massivement d'infections racinaires à Phytophthora en Sardaigne.

## Habitats Naturels

- Maquis méditerranéen
- Zones agroforestières
- Garrigues
- Terrains arides

## Menaces Principales

- Bactérie Xylella fastidiosa
- Phytophthora (clades 2, 6, 8)
- Changement climatique
- Abandon des terrains agricoles

## Lien avec la Biodiversité Locale

Son système racinaire stabilise drastiquement l'érosion des sols tout en abritant la microfaune endémique du bassin méditerranéen.
