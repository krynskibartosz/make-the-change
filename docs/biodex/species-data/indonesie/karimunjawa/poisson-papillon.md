# Poisson-papillon

## Informations de Base

- **Nom français** : Poisson-papillon
- **Nom scientifique** : Chaetodon auriga
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Karimunjawa (Indonésie)

## Description

Poisson omnivore comprimé latéralement avec un faux ocelle distinctif sur la nageoire dorsale. Il broute activement les polypes et contrôle les endoparasites.

## Habitats Naturels

- Récifs coralliens
- Lagons
- Pentés externes
- Zones riches en coraux

## Menaces Principales

- Destructuration des récifs
- Blanchissement corallien
- Perte d'habitat

## Lien avec la Biodiversité Locale

Sa prédation sélective sur les polypes et les petits invertébrés maintient la diversité de l'espace benthique en évitant la domination spatiale d'une seule espèce.
