# Hippocampe

## Informations de Base

- **Nom français** : Hippocampe
- **Nom scientifique** : Hippocampus bargibanti
- **Statut IUCN** : DD
- **Statut IUCN label** : Données insuffisantes
- **Projet associé** : Karimunjawa (Indonésie)

## Description

Minuscule hippocampe pygmée doté d'un mimétisme morphologique parfait avec les gorgones. Extrêmement cryptique, il est sévèrement perturbé par les manipulations des plongeurs.

## Habitats Naturels

- Gorgones du genre Muricella (Muricella plectana, Muricella paraplectana)
- Pentés récifales profondes
- Eaux profondes (20-40 mètres)

## Menaces Principales

- Perturbation par les photographes sous-marins
- Destruction des gorgones hôtes
- Blanchissement corallien

## Lien avec la Biodiversité Locale

Cette espèce témoigne d'un cas extrême de spécialisation symbiotique, intégrant pleinement la structure macroscopique des coraux d'eau profonde.
