# Demoiselle bleue

## Informations de Base

- **Nom français** : Demoiselle bleue
- **Nom scientifique** : Chrysiptera cyanea
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Karimunjawa (Indonésie)

## Description

Petit poisson très agressif et territorial aux teintes bleu vif. Les mâles subissent des changements chromatiques rapides lors de la reproduction.

## Habitats Naturels

- Récifs coralliens
- Lagons
- Zones de forte circulation
- Colonnes d'eau moyennes

## Menaces Principales

- Extraction pour l'aquariophilie
- Destructuration des récifs
- Blanchissement corallien

## Lien avec la Biodiversité Locale

Leur comportement agressif de jardinage benthique influence la structure et la distribution des micro-algues sur le substrat corallien.
