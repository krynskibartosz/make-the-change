# Hérisson européen

## Informations de Base

- **Nom français** : Hérisson européen
- **Nom scientifique** : Erinaceus europaeus
- **Statut IUCN** : NT
- **Statut IUCN label** : Quasi menacé
- **Projet associé** : Habeebee (Belgique)

## Description

Petit mammifère omnivore récemment reclassé à cause de graves déclins démographiques (jusqu'à 50% dans certaines régions). Il est hautement vulnérable à la fragmentation de son territoire.

## Habitats Naturels

- Prairies
- Haies
- Jardins
- Lisières forestières

## Menaces Principales

- Trafic routier
- Perte d'habitat
- Pesticides (anti-limaces)
- Suppression des haies

## Lien avec la Biodiversité Locale

Cet insectivore généraliste orchestre la régulation écologique des macro-invertébrés terrestres, soutenant l'aération et la santé des sols superficiels.
