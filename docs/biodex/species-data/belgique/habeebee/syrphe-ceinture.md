# Syrphe ceinturé

## Informations de Base

- **Nom français** : Syrphe ceinturé
- **Nom scientifique** : Episyrphus balteatus
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Habeebee (Belgique)

## Description

Mouche déguisée en guêpe (mimétisme batésien) et excellent migrateur. Elle présente une écologie duelle fascinante entre son stade larvaire et adulte.

## Habitats Naturels

- Cultures agricoles
- Prairies
- Jardins
- Lisières

## Menaces Principales

- Application de pesticides à large spectre
- Monoculture

## Lien avec la Biodiversité Locale

Sa larve dévore de grandes quantités de pucerons tandis que l'adulte est un pollinisateur généraliste volant sur de longues distances.
