# Papillon citron

## Informations de Base

- **Nom français** : Papillon citron
- **Nom scientifique** : Gonepteryx rhamni
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Habeebee (Belgique)

## Description

Lépidoptère robuste aux ailes mimétiques de feuilles, capable d'hiverner à l'état adulte. Il apparaît aux premiers jours chauds de l'année.

## Habitats Naturels

- Lisières boisées
- Haies
- Jardins
- Vergers

## Menaces Principales

- Perte de plantes hôtes (Rhamnus)
- Pesticides
- Fragmentation

## Lien avec la Biodiversité Locale

Il initie le processus de pollinisation des premières fleurs vernales dès la fin de l'hiver, soutenant la flore précoce.
