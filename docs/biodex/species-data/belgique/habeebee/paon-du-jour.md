# Paon-du-jour

## Informations de Base

- **Nom français** : Paon-du-jour
- **Nom scientifique** : Aglais io
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Habeebee (Belgique)

## Description

Papillon vif orné d'ocelles dissuasives. Ses chenilles dépendent quasi exclusivement de la présence d'orties dioïques pour leur développement.

## Habitats Naturels

- Prairies humides
- Lisières forestières
- Jardins

## Menaces Principales

- Fauchage agricole intensif
- Éradication des orties par herbicides

## Lien avec la Biodiversité Locale

Son abondance et sa grande mobilité assurent une connectivité génétique continue pour les populations isolées de plantes à fleurs.
