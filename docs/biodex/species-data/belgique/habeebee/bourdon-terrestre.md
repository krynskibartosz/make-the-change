# Bourdon terrestre

## Informations de Base

- **Nom français** : Bourdon terrestre
- **Nom scientifique** : Bombus terrestris
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Habeebee (Belgique)

## Description

Grand pollinisateur eusocial à l'efficacité redoutable, souvent commercialisé. Ses ouvrières sont capables d'apprendre les couleurs des fleurs pour optimiser le butinage.

## Habitats Naturels

- Prairies agricoles
- Jardins urbains
- Lisières de bois
- Cultures sous serre

## Menaces Principales

- Pollution par les pesticides
- Perte de ressources florales
- Maladies pathogènes

## Lien avec la Biodiversité Locale

Il offre un service de pollinisation de masse garantissant la reproduction des angiospermes sauvages et le rendement des cultures vivrières.
