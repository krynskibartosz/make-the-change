# Mégachile

## Informations de Base

- **Nom français** : Mégachile
- **Nom scientifique** : Megachile centuncularis
- **Statut IUCN** : LC
- **Statut IUCN label** : Préoccupation mineure
- **Projet associé** : Habeebee (Belgique)

## Description

Abeille solitaire coupeuse de feuilles qui maçonne ses nids au-dessus du sol. Elle subit une forte compétition de la part d'espèces invasives dans les hôtels à insectes.

## Habitats Naturels

- Jardins
- Prairies fleuries
- Zones rudérales

## Menaces Principales

- Intensification agricole
- Compétition par la Mégachile sculptée invasive

## Lien avec la Biodiversité Locale

Elle intègre physiquement le matériel végétal dans la structure du sol ou des cavités tout en transférant le pollen des fleurs sauvages.
