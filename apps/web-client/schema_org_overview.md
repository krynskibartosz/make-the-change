# Schema.org Overview

## About Schema.org
**Source:** [https://schema.org/docs/about.html](https://schema.org/docs/about.html)

Schema.org is a collaborative, community activity with a mission to create, maintain, and promote schemas for structured data on the Internet.
*   **Mission:** To improve the web by creating a structured data vocabulary that is supported by major search engines (Google, Microsoft, Yahoo, Yandex).
*   **Governance:** Led by a Steering Group with representatives from founding companies and the W3C.
*   **Community:** Open community process via W3C Community Group and GitHub.

## Homepage & Navigation
**Source:** [https://schema.org](https://schema.org)

The homepage serves as the central hub for:
*   **Docs:** Access to full documentation.
*   **Schemas:** The vocabulary hierarchy.
*   **Validate:** Link to the [Schema Validator](https://validator.schema.org).
*   **Usage:** Used by over 45 million web domains (as of 2024) to markup content for rich experiences (snippets, knowledge graphs).

## Documentation Resources
**Source:** [https://schema.org/docs/documents.html](https://schema.org/docs/documents.html)

Key resources available:
*   **Getting Started:** Introduction to microdata and schema usage.
*   **Schemas:** The full hierarchy of types.
*   **Data Model:** Technical details on the RDF/RDFS-based data model.
*   **Developers:** Technical resources for implementers.
*   **Releases:** History of schema updates.
