🚀 Stratégie d'Onboarding & Monétisation (Upsell)

1. Le Paradigme de l'Onboarding (Web App)

L'application étant web-based avec une Landing Page riche (Gradual Engagement), l'onboarding éducatif classique (écrans à swiper) est supprimé pour éviter la redondance.

Le Tunnel d'Onboarding Actif :
L'onboarding est fusionné avec la création de compte. Il se concentre sur l'engagement émotionnel :

Clic sur "Créer mon compte" (depuis un projet ou un défi).

Écran "Choisis ton compagnon" (Melli, Sylva, Aura). -> Attachement émotionnel.

Saisie de l'email (Magic Link) ou Google/Apple Sign-in.

Atterrissage direct dans le BioDex ou le Collectif.

2. Cartographie des Points d'Entrée Premium (Le Paywall "Gardiens")

Pour maximiser la conversion (viser > 5% d'utilisateurs gratuits vers payants), le Paywall ne doit pas être caché dans les paramètres. Il doit intercepter l'utilisateur à des moments stratégiques.

Point d'entrée A : L'Upsell d'Onboarding (Moment de haute motivation)

Quand : Immédiatement après la création du compte et le choix du compagnon, juste avant la première vraie session de jeu.

UX : Afficher l'écran "Devenez un Gardien" avec un texte contextuel : "Super choix ! En tant que Gardien, vos actions pour [Nom de la Faction] auront deux fois plus d'impact dès aujourd'hui."

Sortie : L'utilisateur peut souscrire (essai de 7 jours) ou fermer via la croix (X) pour continuer gratuitement.

Point d'entrée B : Le "Friction-Point" Contextuel (La frustration positive)

Quand : Pendant l'utilisation quotidienne de l'application gratuite.

UX : Intégrer des éléments visuels Premium (avec l'icône Couronne 👑 ou la couleur vert néon) dans l'interface gratuite.

Exemple 1 : Dans l'onglet Défis, un bouton grisé Boost Gardien (x2) 👑. Au clic -> Ouverture du Paywall.

Exemple 2 : Dans le Marché, certains produits rares ont un tag "Accès anticipé Gardiens". Au clic -> Ouverture du Paywall.

Point d'entrée C : La Visibilité Passive (Le Profil)

Quand : En permanence.

UX : Dans l'onglet Profil de l'utilisateur gratuit, sous les 4 KPIs principaux, ajouter une carte ou une bannière call-to-action : ⭐ Débloquez l'Aura du Gardien.

Point d'entrée D : La Gestion (Les Paramètres)

Quand : Uniquement quand l'utilisateur cherche à gérer sa facturation.

UX : Le bouton "Abonnement & Avantages" dans les paramètres (Écran 63). S'il est premium, il gère son offre. S'il est gratuit, il voit le Paywall classique. Ce n'est pas un outil d'acquisition principal.
